### Avatars List

```jsx
<Avatar.List>
  <Avatar imageURL="/demo/faces/male/9.jpg" />
  <Avatar imageURL="/demo/faces/female/8.jpg" />
  <Avatar imageURL="/demo/faces/male/4.jpg" />
</Avatar.List>
```

### Stacked List

```jsx
<Avatar.List stacked>
  <Avatar imageURL="/demo/faces/male/9.jpg" />
  <Avatar imageURL="/demo/faces/female/8.jpg" />
  <Avatar imageURL="/demo/faces/male/4.jpg" />
  <Avatar>+8</Avatar>
</Avatar.List>
```
