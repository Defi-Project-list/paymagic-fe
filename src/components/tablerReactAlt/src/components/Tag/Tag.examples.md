```jsx
<Tag.List>
  <Tag>First tag</Tag>
  <Tag>Second tag</Tag>
  <Tag>Third tag</Tag>
</Tag.List>
```

### Link tag

```jsx
<Tag.List>
  <Tag link href="">
    First tag
  </Tag>
  <Tag link href="">
    Second tag
  </Tag>
  <Tag link href="">
    Third tag
  </Tag>
</Tag.List>
```

### Rounded tag

```jsx
<Tag.List>
  <Tag rounded>First tag</Tag>
  <Tag rounded>Second tag</Tag>
  <Tag rounded>Third tag</Tag>
</Tag.List>
```

### Color tag

```jsx
<Tag.List>
  <Tag color="blue">Blue</Tag>

```

### Tag remove

Adds an 'x' onthe right side of the Tag

```jsx
<Tag.List>
  <Tag remove>First tag</Tag>
  <Tag remove>Second tag</Tag>
  <Tag remove>Third tag</Tag>
  <Tag remove>Fourth tag</Tag>
</Tag.List>
```
